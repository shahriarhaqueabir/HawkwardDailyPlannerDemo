\# Daily Planner Pro



A lightweight, offline-first daily planner built as a single-file web app.



The app focuses on clarity and speed:

\- Kanban-style task management

\- Drag \& drop workflow

\- Progress dashboard

\- Visual reports

\- No backend, no accounts, no setup



\## Features

\- Three-column task board (To Do / In Progress / Done)

\- Persistent storage using browser localStorage

\- Dashboard with completion stats

\- Reports view with task distribution chart

\- Works entirely offline



\## Usage

Open `index.html` in any modern browser.



No build step required.



\## Tech Stack

\- Vanilla JavaScript

\- Tailwind CSS (CDN)

\- Chart.js



\## Design Principles

\- Simplicity over abstraction

\- Explicit state management

\- Single-file architecture for ease of distribution
